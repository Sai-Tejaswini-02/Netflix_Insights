# Netflix Content Insights Dashboard

## Project Overview

This project is a Power BI dashboard that analyzes Netflix content data. It shows insights about movies and TV shows, genres, ratings, and trends over time.

## Tools Used

* Power BI
* Excel / CSV

## Dataset

The dataset contains Netflix data such as:

* Title
* Type (Movie / TV Show)
* Genre
* Country
* Release Year
* Rating

## Dashboard Features

* Movies vs TV Shows analysis
* Country-wise content distribution
* Genre analysis
* Content trends over the years
* Filters for easy exploration

## Key Insights

* Movies are more than TV shows on Netflix
* Drama and comedy are the most common genres
* Content has increased in recent years
